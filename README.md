# 🧮 Calculator App

A sleek, responsive, and user-friendly web-based calculator built using modern web technologies. 

🌐 **[Live Demo](https://polojuarunkumar.github.io/Calculator/)**

---

## 🚀 Features

* **Basic Arithmetic:** Supports addition, subtraction, multiplication, and division.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens.
* **Clear & Delete:** Easily clear the entire screen (`C`) or backspace the last digit (`⌫`).
* **Keyboard Support:** Input values directly using your physical keyboard.

---

## 🛠️ Technologies Used

* **HTML5:** For semantic structure.
* **CSS3:** For modern layout, styling, and responsiveness.
* **JavaScript (ES6):** For core mathematical logic and button functionality.
